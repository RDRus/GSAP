Voortgang
🟢 Niveau 1 — Basis (instap)

Laat een box
naar rechts bewegen

Jouw eerste gsap.to() animatie in actie.

Maak een animatie waarbij een vierkant (<div>) naar rechts beweegt op de pagina.

⚙️ Voeg GSAP toe aan je project

Plaats de GSAP-bibliotheek via CDN binnen de <head> van je HTML-document:

<script src="https://cdn.jsdelivr.net/npm/gsap@3.14.2/dist/gsap.min.js"></script>
💡 HTML 
<div class="box"></div>
💡 CSS 
.box {
  width: 100px;
  height: 100px;
  background-color: blue;
}
💡 GSAP 
gsap.to(".box", {
  x: 200,
  duration: 2
});
✅ Wat leer je hier?
  • wat gsap.to() doet
  • wat x betekent
  • wat duration doet

🟡 Niveau 2 — Gemiddeld

Wachten, bewegen
& verdwijnen

Combineer meerdere properties in één animatie.

Laat de box:

  • eerst wachten
  • dan naar rechts bewegen
  • én langzaam verdwijnen
⚙️ Voeg GSAP toe aan je project

Plaats de GSAP-bibliotheek via CDN binnen de <head> van je HTML-document:

<script src="https://cdn.jsdelivr.net/npm/gsap@3.14.2/dist/gsap.min.js"></script>
💡 Hint
Gebruik de volgende properties:

delay — laat de animatie later starten

opacity — maak het element (on)zichtbaar
💡 Voorbeeldoplossing 
gsap.to(".box", {
  x: 200,
  duration: 2,
  delay: 1,
  opacity: 0
});
✅ Wat leer je hier?
  • meerdere properties combineren
  • timing controleren met delay
  • visuele effecten toevoegen

🔴 Niveau 3 — Uitdaging

Twee stappen,
één animatie

Sequencing en timelines — voor de gevorderden.

Maak een animatie met 2 stappen:

  • De box beweegt naar rechts
  • Daarna gaat hij naar beneden

👉 Bonus: Laat hem ook van kleur veranderen

⚙️ Voeg GSAP toe aan je project

Plaats de GSAP-bibliotheek via CDN binnen de <head> van je HTML-document:

<script src="https://cdn.jsdelivr.net/npm/gsap@3.14.2/dist/gsap.min.js"></script>
💡 Zonder timeline 
gsap.to(".box", {
  x: 200,
  duration: 2
});

gsap.to(".box", {
  y: 150,
  duration: 2,
  delay: 2
});
💡 Met timeline (challenge) 
let tl = gsap.timeline();

tl.to(".box", {
  x: 200,
  duration: 2
});

tl.to(".box", {
  y: 150,
  duration: 2
});
🎨 Bonus — Kleur veranderen 
gsap.to(".box", {
  x: 200,
  duration: 2,
  backgroundColor: "red"
});
✅ Wat leer je hier?
  • volgorde (sequencing)
  • meerdere animaties combineren
  • verschil tussen delay en timeline
